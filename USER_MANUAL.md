## 📊 Dashboard

หน้าหลักแสดงข้อมูลสรุปของระบบ:

| Widget | รายละเอียด |
|--------|-----------|

## 📦 จัดการสินค้า

### เพิ่มสินค้าใหม่
1. ไปที่ **สินค้า** > **เพิ่มสินค้า**
2. กรอกข้อมูล:
   - รหัสสินค้า
   - ชื่อสินค้า
   - หมวดหมู่
   - หน่วย
   - ราคา/หน่วย
   - Safety Stock
3. คลิก **บันทึก**

### แก้ไขสินค้า
1. คลิกที่สินค้าในรายการ
2. คลิก **แก้ไข**
3. แก้ไขข้อมูล
4. คลิก **บันทึก**

### ค้นหาสินค้า
- ใช้ช่องค้นหาด้านบน
- หรือกด `Ctrl+K` เพื่อค้นหาทั่วระบบ

### พิมพ์ฉลากสินค้า
1. เปิดหน้าสินค้า
2. คลิก **พิมพ์ฉลาก**
3. เลือกขนาดและจำนวน
4. คลิก **พิมพ์**

---

## 📥 เคลื่อนไหวสต็อก

### รับสินค้าเข้า (Stock In)
1. ไปที่ **เคลื่อนไหว** > **รับเข้า**
2. เลือกสินค้า
3. ใส่จำนวน
4. ระบุหมายเหตุ
5. คลิก **บันทึก**

### เบิกสินค้าออก (Stock Out)
1. ไปที่ **เคลื่อนไหว** > **เบิกออก**
2. เลือกสินค้า
3. ใส่จำนวน
4. ระบุเหตุผล/ผู้เบิก
5. คลิก **บันทึก**

### ปรับสต็อก (Adjustment)
1. ไปที่ **ปรับสต็อก**
2. เลือกสินค้า
3. ใส่จำนวนใหม่ หรือ +/- adjustment
4. ระบุเหตุผล
5. คลิก **บันทึก**

---

## 🛒 ใบสั่งซื้อ (PO)

### สร้าง PO ใหม่
1. ไปที่ **ใบสั่งซื้อ** > **สร้างใหม่**
2. เลือกผู้ขาย
3. เพิ่มรายการสินค้า
4. กรอกจำนวนและราคา
5. คลิก **บันทึก**

### สถานะ PO
| สถานะ | ความหมาย |
|-------|---------|
| 📝 Draft | ร่าง |
| ⏳ Pending | รออนุมัติ |
| ✅ Approved | อนุมัติแล้ว |
| 📦 Ordered | สั่งซื้อแล้ว |
| ✔️ Received | รับสินค้าแล้ว |

### อนุมัติ PO
1. เปิด PO
2. ตรวจสอบรายการ
3. ลงนาม (Digital Signature)
4. คลิก **อนุมัติ**

---

## 🔄 ยืม-คืน

### สร้างรายการยืม
1. ไปที่ **ยืม-คืน** > **ยืมใหม่**
2. กรอกชื่อผู้ยืม
3. เลือกสินค้าและจำนวน
4. กำหนดวันที่ต้องคืน
5. คลิก **บันทึก**

### คืนสินค้า
1. เปิดรายการยืม
2. คลิก **คืน**
3. ตรวจสอบสภาพสินค้า
4. กรอกหมายเหตุ (ถ้ามี)
5. คลิก **ยืนยันคืน**

---

## 📋 ตรวจนับสต็อก

### สร้างรายการตรวจนับ
1. ไปที่ **ตรวจนับสต็อก** > **สร้างใหม่**
2. เลือกคลังสินค้า
3. กำหนดวันที่
4. คลิก **สร้าง**

### ทำการตรวจนับ
1. คลิก **เริ่มตรวจนับ**
2. กรอกจำนวนที่นับได้ในแต่ละรายการ
3. ระบบจะคำนวณผลต่างให้อัตโนมัติ
4. เสร็จแล้วคลิก **บันทึกผล**

## 📊 รายงาน

### ABC Analysis
- วิเคราะห์สินค้าตามหลัก Pareto (80/20)
- **Class A**: สินค้าสำคัญมาก (20% ของรายการ = 80% ของมูลค่า)
- **Class B**: สินค้าสำคัญปานกลาง
- **Class C**: สินค้าสำคัญน้อย

### ROI Calculator
1. ไปที่ **รายงาน** > **ROI Calculator**
2. กรอกราคาซื้อ/ขาย
3. กรอกจำนวนและต้นทุนเก็บรักษา
4. ระบบคำนวณ ROI อัตโนมัติ

### Custom Report Builder
1. เลือกประเภทข้อมูล
2. เลือกคอลัมน์ที่ต้องการ
3. เพิ่มตัวกรอง
4. คลิก **รัน**
5. Export เป็น CSV
